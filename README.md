**📝 SwiftUI Todo App**

A simple and clean Todo application built using SwiftUI.

This project focuses on understanding state management, user input handling, and dynamic UI updates in iOS development.

**🚀 Features**

✅ Add new tasks

🗑 Delete tasks instantly

⌨️ Enter key support

🔄 Real-time UI updates using SwiftUI state

🎨 Clean and minimal UI

⚙️ CI/CD integration with GitHub Actions

**🛠 Tech Stack**

Swift

SwiftUI

Xcode

GitHub Actions (CI/CD)

**📦 How It Works**

The app uses:

@State for managing local view state

@Binding for passing data between views

ForEach for rendering dynamic task lists

SwiftUI’s reactive UI system for automatic updates

When a task is added or deleted, the UI updates automatically based on state changes.

**🔄 CI/CD Implementation**

This project includes a basic CI/CD pipeline using GitHub Actions.

Workflow includes:

🔹 Automatic build on every push

🔹 Run unit tests

📸 Screenshots

🔹 Track workflow status

🔹 Ensure build stability before merging

This helps simulate a real-world development workflow and ensures code quality.
